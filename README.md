# vertx-clj

A clojure wrapper for vertx development.

## Usage

Add [vertx-clj "0.1.1-SNAPSHOT"] to your project.clj

Checkout [examples
directory](https://github.com/isaiah/vertx-clj/tree/master/examples) for
usage.
